# HR Dashboard (Streamlit)

## Features
- Attendance tracking
- Leave management
- Shift scheduling
- Login system
- Filters + charts
- CSV export

## Virtual Environment
python -m venv venv
source venv/bin/activate   # (Linux/Mac)
venv\Scripts\activate      # (Windows)

## Installation
pip install -r requirements.txt

## Run
streamlit run app.py

## Snapshot of packages
pip freeze > requirements.txt
